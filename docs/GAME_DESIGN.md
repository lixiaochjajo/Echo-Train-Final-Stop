# 《回声列车：终点站》游戏设计目标

## 1. 一句话定位

一款以固定空间探索为骨架、以死亡回退和世界记忆为核心的魔塔式 Roguelite：玩家会保留成长，列车也会保留玩家死亡造成的回声。

## 2. 世界设定

玩家被困在一列拥有 50 节车厢的异常列车中。每节车厢相当于一层，包含战斗、乘客聚落、商店、机关、对话、隐藏区域和 Boss。

每十节车厢进入一个大型站台。玩家从多个候选地点中选择一个记忆锚点，死亡后回到锚点，但部分成长、情报与世界异常会保留。

## 3. 核心体验

核心循环：

```text
探索车厢
→ 战斗 / 对话 / 交易
→ 获取装备和战术卡
→ 选择路线与资源分配
→ 绑定记忆锚点
→ 死亡
→ 继承等级、情报与一项遗产
→ 世界生成回声变化
→ 使用新解法再次探索
```

希望玩家形成的感觉：

> 我不是简单重来。我知道得更多，而这个世界也因为我的死亡变得不一样了。

## 4. 地图结构

采用“宏观路线 + 微观方块地图”的混合结构。

### 宏观层

每十层是一张分支路线图，负责战略取舍：

- 安全路线：补给稳定，奖励一般
- 冲突路线：战斗密集，装备更好
- 异常路线：事件与回声较多，风险不可预测

宏观路线不会每次完全随机重置。死亡后在原路线基础上新增支线、替换少量节点或显示已知情报。

### 微观层

进入节点后使用魔塔式方块地图，重点承载：

- 空间记忆
- 门锁与机关
- 敌人位置
- NPC 状态
- 隐藏墙与捷径
- 上一轮死亡位置
- 可绕过或稍后处理的强敌

死亡后主地图保持可识别，只增加新的解决方式。

## 5. 记忆锚点

每十层提供多个候选存档点，但只能绑定一个。

候选类型：

### 稳定锚点

- 复活时补给较多
- 世界异常增长较慢
- 稀有奖励较少

### 贪婪锚点

- 可以额外保留装备
- 下一轮敌人更强
- 稀有掉落提高

### 记忆锚点

- 保留更多情报或卡牌
- 解锁特殊对话
- 复活资源较少

锚点位置也需要权衡：早期锚点安全但重复路程长，后期锚点省时间但复活状态更危险。

## 6. 死亡与继承

### 必定保留

- 等级
- 已解锁技能
- 敌人图鉴
- 事件情报
- 隐藏位置记录
- 关键剧情记忆

### 死亡遗产二选一

#### 装备铭刻

从身上的装备中保留一件。传奇装备可以损失词条或累积副作用，防止永久携带一件毕业装备。

#### 命运抽取

放弃装备保留，根据推进距离、精英击杀、装备价值、隐藏探索和等级差获得一次三选一奖励。

可能奖励：

- 遗物
- 稀有卡牌
- 下一轮临时增益
- 特殊事件钥匙
- 制造材料
- 修改一个车厢状态的机会

## 7. 等级与难度

等级保留，但不能通过无限死亡把所有内容纯数值碾压。

等级主要提供：

- 少量基础属性
- 卡牌槽位
- 被动技能点
- 对话能力
- 装备适应性
- 战斗干预次数
- 更完整的敌人信息

旧敌人允许被等级压制，让成长有真实反馈；后期困难更多来自机制而非单纯生命膨胀。

机制示例：

- 护盾必须破甲
- 敌人复制最高伤害牌
- 死亡时强化同伴
- 限定轮数
- 持续消耗食物或理智
- Boss 封印高频装备

## 8. 非线性难度

每十层使用波浪形难度：

| 楼层阶段 | 体验目标 |
| --- | --- |
| 1–3 | 探索、补给、认识新机制 |
| 4 | 第一次强度峰值 |
| 5 | 安全区或大型剧情 |
| 6–7 | 路线取舍与资源消耗 |
| 8 | 机制型精英 |
| 9 | 高风险支线 |
| 10 | Boss 与锚点选择 |

困难来源轮换：战斗、资源、路线判断、克制关系、对话代价、时间限制、护送、连续战斗和无法恢复。

## 9. 固定事件如何避免重复

采用“主干固定，状态变化”。同一个事件保存多个阶段：

- 初次相遇
- 已知结果
- 曾经被背叛
- 掌握关键证据
- NPC 产生既视感
- 高回声状态

玩家死亡后再来，可以提前揭露、布局陷阱、绕过过程、改变交易条件或进入上一轮不存在的隐藏区域。

避免大量纯概率检定。对话更强调条件和代价：身份、证据、装备、生命上限、卡牌或后续关系变化。

## 10. 回声系统

每次死亡记录原因标签：

- 中毒
- Boss 处决
- 资源耗尽
- NPC 背叛
- 战斗超时
- 诅咒爆发
- 装备损坏

下一轮根据标签生成一至两个回声事件。

例如中毒死亡后可能出现医生、解毒室、毒素研究卡、更强毒巢或由尸体形成的毒雾。

上一轮尸体会留在世界中，可以：

- 取回未保留装备
- 与过去自己的影子战斗
- 查看死亡前记录
- 获得一次性回声卡
- 开启隐藏通道
- 被敌人利用并变成精英
- 被 NPC 发现并改变立场

## 11. 回声等级

死亡增加回声等级，用于控制异常程度，而非简单全局难度缩放。

### 低回声

世界稳定，隐藏内容较少。

### 中回声

出现旧尸体、NPC 既视感、路线变化和异常精英。

### 高回声

不同时间线重叠，已死 NPC 重新出现，同一车厢存在多个版本，Boss 开始记住玩家。

每十层设置异常上限，避免死亡过多导致系统失控。

## 12. 战斗系统

采用“战术卡选择 + 自动战斗”。

战斗前配置：

- 装备
- 6–10 张战术卡
- 触发优先级
- 站位
- 目标规则

战斗中角色自动攻击并积累能量，玩家在关键时刻使用：

- 打断
- 闪避
- 换位
- 治疗
- 复制
- 反击
- 召唤
- 撤退

普通战斗目标时长 15–30 秒，精英和 Boss 增加更多人工干预。

卡牌分为：

- 战斗牌
- 自动战斗规刑牌
- 探索牌
- 对话牌
- 死亡后获得的回声牌

## 13. Boss 学习玩家

Boss 根据上一轮策略获得针对能力：

- 针对护盾流获得破盾
- 针对中毒流携带解毒
- 针对召唤流优先攻击召唤物
- 临时封印玩家最依赖的装备

玩家也获得 Boss 情报，可提前查看技能、识别弱点、破坏装置或拉拢手下。

双方都在学习，形成宿敌感。

## 14. 五章结构

1. **1–10：失控列车**——生存、基础战斗与第一次死亡
2. **11–20：乘客社会**——阵营、交易、谎言与背叛
3. **21–30：机械车间**——装备改造、规则卡和机关
4. **31–40：时间重叠区**——多版本地图与明显回声变化
5. **41–50：列车核心**——Boss 与 NPC 认识到循环，处理终止轮回的代价

## 15. 第一阶段验证目标

完整内容扩展前，先验证：

1. 自动战斗加战术卡是否有操作感
2. 死亡后重新探索是否仍有新鲜感
3. 保留等级后是否仍有策略压力
4. 锚点与遗产选择是否足够纠结
5. 固定地图上的新支线是否比完全随机地图更有记忆价值

当前仓库的第一版是单节车厢垂直切片，用来验证最小死亡循环。

## 17. 原型 0.2 验证内容

0.2 不增加大量地图面积，优先验证死亡循环是否成立：

1. 战斗前展示三种战术的确定性预估，玩家可以明确承担风险。
2. 精英会记住上一轮使用的战术，重复解法不再始终最优。
3. 死亡后必须选择“铭刻装备”或“命运抽取”，不能同时获得。
4. 命运奖励参考死亡压力和被放弃装备的价值，而非完全均匀随机。
5. 同一道隔离门可以用永久情报、当前生命或回声等级解决。
6. NPC 在玩家死亡后出现新的对话方式，固定地图因此产生新内容。

该版本的核心观察指标：玩家是否愿意为了获得协议情报主动改变路线，以及死亡后是否会认真比较装备和永久奖励，而不是无脑选择固定项。
